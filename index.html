HTML Tags

In website we have different portion mention below

  1. Header:

    2. Defines a header for a document or a section

  2. Navigation:

    3. Defines a set of navigation links

  3. Side Portion

    4. Defines content aside from the content (like a sidebar)

  4. Section

    5. Defines a section in a document

  5. Article

    6. Defines an independent, self-contained content

  6. Main part

    7. This tag used to show the main part of the website

  7. Footer:

    8. Defines a footer for a document or a section
